gplus:addTestStubSources

Full name:

org.codehaus.gmavenplus:gmavenplus-plugin:1.13.0-SNAPSHOT:addTestStubSources

Description:

Adds Groovy test stubs directory back to Maven's list of test source directories. Normally, you won't need to use this mojo.

Attributes:

  • Requires a Maven project to be executed.
  • Since version: 1.1.

Optional Parameters

Name Type Since Description
<skipTests> boolean 1.1 Flag to allow adding test sources to be skipped.
Default value is: false.
User property is: maven.test.skip.
<testStubsOutputDirectory> File 1.1 The location for the compiled test classes.
Default value is: ${project.build.directory}/generated-sources/groovy-stubs/test.

Parameter Details

<skipTests>

Flag to allow adding test sources to be skipped.
  • Type: boolean
  • Since: 1.1
  • Required: No
  • User Property: maven.test.skip
  • Default: false

<testStubsOutputDirectory>

The location for the compiled test classes.
  • Type: java.io.File
  • Since: 1.1
  • Required: No
