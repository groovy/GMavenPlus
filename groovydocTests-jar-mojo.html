gplus:groovydocTests-jar

Full name:

org.codehaus.gmavenplus:gmavenplus-plugin:4.0.1:groovydocTests-jar

Description:

Create a GroovyDoc jar for the test sources. Note by default this will also invoke the groovydocTests goal (unless invokeGroovyDoc is false).

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: test.
  • The goal is thread-safe and supports parallel builds.
  • Since version: 1.7.1.
  • Binds by default to the lifecycle phase: package.
