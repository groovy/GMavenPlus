gplus:addSources

Full name:

org.codehaus.gmavenplus:gmavenplus-plugin:2.1.0:addSources

Description:

This mojo adds Groovy sources to the project's sources.

Attributes:

  • Requires a Maven project to be executed.
  • Since version: 1.0-beta-1.
  • Binds by default to the lifecycle phase: initialize.

Optional Parameters

Name Type Since Description
<sources> FileSet[] 1.0-beta-1 The Groovy source files (relative paths). Default: "${project.basedir}/src/main/groovy/**/*.groovy"
