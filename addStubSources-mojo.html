gplus:addStubSources

Full name:

org.codehaus.gmavenplus:gmavenplus-plugin:1.8.0:addStubSources

Description:

Adds Groovy stubs directory back to Maven's list of source directories. Normally, you won't need to use this mojo.

Attributes:

  • Requires a Maven project to be executed.
  • Since version: 1.1.

Optional Parameters

Name Type Since Description
<stubsOutputDirectory> File 1.1 The location for the compiled classes.
Default value is: ${project.build.directory}/generated-sources/groovy-stubs/main.

Parameter Details

<stubsOutputDirectory>

The location for the compiled classes.
  • Type: java.io.File
  • Since: 1.1
  • Required: No
  • Default: ${project.build.directory}/generated-sources/groovy-stubs/main
